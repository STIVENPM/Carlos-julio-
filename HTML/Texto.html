Jerarquia de titulos

En HTML, la jerarquía se implementa mediante las etiquetas de encabezado que van desde h1 hasta h6, donde el número indica el nivel de importanciay la profundidad en el esquema del documento. El h1 es el elemento de mayor rango (generalmente reservado para el título único de la página), mientras que del h2 al h6 se utilizan para dividir el contenido en secciones y subsecciones cada vez más específicas. Es fundamental seguir un orden descendente sin saltarse niveles (por ejemplo, no pasar de un h2 a un h4), ya que esto permite que los lectores de pantalla y los motores de búsqueda interpreten correctamente la estructura semántica y la relación de dependencia entre los temas.

Formato de Texto Básico

El formato de texto básico en HTML se centra en aportar semántica y estructura a las palabras, permitiendo que el navegador y el usuario distingan la intención detrás de cada fragmento como :

  • <p>

    • se usa para que agrupa bloques de texto con un espacio intrínseco.

  • <em>

    • para dar énfasis gramatical (cursiva).

  • <br>

    • Se usa para saltos de línea simples.

Enlaces

Tipos de enlaces:

Los enlaces en HTML <a> conectan recursos mediante el atributo href, dividiéndose principalmente en internos (mismo sitio), externos (sitios terceros), y anclas (posiciones dentro de una misma página)

Enlace Externo:

Es un enlace que lleva a otra página web diferente (otro sitio de internet) ejemplo:

Ir a Google

Sintaxis:

<a href="enlace">

Enlace Interno:

Es un enlace que lleva a otra página dentro del mismo proyecto ejemplo:

Ir a Inicio

Sintaxis:

<a href="index.html">

Insertar un sitio web:

Sintaxis:

<iframe href="enlace">
